# Extra Animations

This mod aims to supplement and improve the vanilla animations in third-person gameplay. Some animation examples are shown below.

[![Buy Me a Coffee at ko-fi.com](https://i.postimg.cc/SQ5ZLKg5/support-me-on-kofi-beige.png)](https://ko-fi.com/X8X71FI3YO)

## Compass

Activated by holding compasses

![Compass](https://i.postimg.cc/yN1hN9Gx/compass.gif)

![Throw](https://i.postimg.cc/dQb8ygfb/throw.gif)

![Brush](https://i.postimg.cc/SswzczZp/brush.gif)

![Magma](https://i.postimg.cc/d32DdNk3/magma.gif)

## Levitation

![Levitation](https://i.postimg.cc/15Kf3T1L/levitation.gif)
